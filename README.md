Project Outline:
This project will allow you to play Mp3 songs, It will utilize an Esp32 dev board in tandem with a sd car breakout, a PCM5102, and a MAX97220. 
The user will interact with the device via a rotary encoder, a mechanical key switch, and an oled display. The audio will play out of a pair of
over ear headphones. The system will be built on perf-board with elegant hand routed wires.
